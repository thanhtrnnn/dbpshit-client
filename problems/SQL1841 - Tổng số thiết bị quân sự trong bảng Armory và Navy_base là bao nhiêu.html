Mô tả bảng

Bảng armory

Lưu trữ thông tin về các trang bị quân sự trong kho.

+------------+----------------+
| Cột        | Kiểu dữ liệu   |
+------------+----------------+
| id         | INT            |
| equipment  | VARCHAR(255)   |
+------------+----------------+

Bảng naval_base

Lưu trữ thông tin về các trang bị trong các căn cứ hải quân.

+------------+----------------+
| Cột        | Kiểu dữ liệu   |
+------------+----------------+
| id         | INT            |
| equipment  | VARCHAR(255)   |
+------------+----------------+

Nhiệm vụ

Liệt kê tổng số thiết bị quân sự trong mỗi bảng armory và naval_base.

Input mẫu

Bảng armory

+----+-------------+
| id | equipment   |
+----+-------------+
| 1  | Tank        |
| 2  | Artillery   |
| 3  | Rifle       |
+----+-------------+

Bảng naval_base

+----+-------------+
| id | equipment   |
+----+-------------+
| 1  | Submarine   |
| 2  | Carrier     |
| 3  | Destroyer   |
| 4  | Frigate     |
+----+-------------+

Output mẫu

+----------+
| COUNT(*) |
+----------+
| 3        |
| 4        |
+----------+
