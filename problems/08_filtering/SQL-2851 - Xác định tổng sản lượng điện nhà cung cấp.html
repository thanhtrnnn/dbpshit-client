ID: SQL-2851 Tiêu đề: Xác định tổng sản lượng điện nhà cung cấp URL WEB: https://db.ptit.edu.vn/question-detail/26a87b89-4f53-4fd8-b2c4-817de6cba96a URL API: https://dbapi.ptit.edu.vn/api/app/question/26a87b89-4f53-4fd8-b2c4-817de6cba96a Loại Database: Mysql ------------------------------

Hệ thống theo dõi sản xuất điện từ các nguồn tái tạo theo vùng và nhà cung cấp. CSDL của hệ thống bao gốm các bảng sau:

Bảng region – Quản lý thông tin về các vùng địa lý

• `region (id INT PRIMARY KEY, name VARCHAR(50))`:

• `id (INT)`: Mã vùng.

• `name (VARCHAR(50))`: Tên vùng.

Bảng provider – Quản lý thông tin về các nhà cung cấp năng lượng

• `provider (id INT PRIMARY KEY, name VARCHAR(50), region_id INT)`:

• `id (INT)`: Mã nhà cung cấp.

• `name (VARCHAR(50))`: Tên nhà cung cấp.

• `region_id (INT)`: Vùng mà nhà cung cấp hoạt động.

Bảng renewable_source – Quản lý thông tin về các nguồn năng lượng tái tạo

• `renewable_source (id INT PRIMARY KEY, name VARCHAR(50), provider_id INT)`:

• `id (INT)`: Mã nguồn năng lượng tái tạo.

• `name (VARCHAR(50))`: Tên nguồn (ví dụ: "Solar" là năng lượng mặt trời, "Wind" là năng lượng gió ...).

• `provider_id (INT)`: Nhà cung cấp sử dụng nguồn năng lượng này.

Bảng power_generation – Quản lý thông tin về sản lượng điện sản xuất

• `power_generation (id INT PRIMARY KEY, date DATE, source_id INT, power_generated INT)`:

• `id (INT)`: Mã bản ghi sản lượng.

• `date (DATE)`: Ngày sản xuất.

• `source_id (INT)`: Mã nguồn năng lượng sử dụng.

• `power_generated (INT)`: Lượng điện được sản xuất (đơn vị kWh hoặc tương đương).


Nhiệm vụ

Xác định 3 nhà cung cấp năng lượng gió (Wind) có tổng sản lượng điện lớn nhất trong các vùng có tên bắt đầu bằng chữ ‘S’ (ví dụ: “South”, “Southeast”) trong khoảng thời gian từ 2021-04-01 đến 2021-06-30
