ID: SQL1617 Tiêu đề: Tên và ngày phóng của vệ tinh được phóng gần đây nhất của mỗi nhà sản xuất là gì? URL WEB: https://db.ptit.edu.vn/question-detail/b08967f4-6f48-4d82-a74c-e5cdd24a5e30 URL API: https://dbapi.ptit.edu.vn/api/app/question/b08967f4-6f48-4d82-a74c-e5cdd24a5e30 Loại Database: Sql Server ------------------------------

Mô tả bảng

Bảng manufacturers

+-----------+-----------+
| Cột       | Kiểu dữ liệu |
+-----------+-----------+
| id        | INT       |
| name      | TEXT      |
+-----------+-----------+

Bảng satellites

+------------------+-------------+
| Cột              | Kiểu dữ liệu |
+------------------+-------------+
| id               | INT         |
| manufacturer_id  | INT         |
| name             | TEXT        |
| launch_date      | DATE        |
+------------------+-------------+

Nhiệm vụ

Tìm ra vệ tinh được phóng gần nhất (có launch_date lớn nhất) của mỗi nhà sản xuất, đồng thời hiển thị:

  • Tên nhà sản xuất
  • Tên vệ tinh
  • Ngày phóng vệ tinh

Input mẫu

Bảng manufacturers

+----+---------------+
| id | name          |
+----+---------------+
| 1  | SpaceX        |
| 2  | NASA          |
| 3  | Roscosmos     |
+----+---------------+

Bảng satellites

+----+------------------+-------------+-------------+
| id | manufacturer_id  | name        | launch_date |
+----+------------------+-------------+-------------+
| 1  | 1                | FalconSat   | 2020-01-10  |
| 2  | 1                | Starlink-5  | 2024-05-22  |
| 3  | 2                | Hubble      | 1990-04-24  |
| 4  | 2                | Artemis-1   | 2022-11-16  |
| 5  | 3                | Luna-25     | 2023-08-11  |
+----+------------------+-------------+-------------+

Output mẫu

+---------------+-------------+-------------+
| manufacturer  | satellite   | launch_date |
+---------------+-------------+-------------+
| SpaceX        | Starlink-5  | 2024-05-22  |
| NASA          | Artemis-1   | 2022-11-16  |
| Roscosmos     | Luna-25     | 2023-08-11  |
+---------------+-------------+-------------+
