ID: SQL1829 Tiêu đề: Tổng ngân sách dành cho các sáng kiến ​​phân chia kỹ thuật số của các công ty trong khu vực công là bao nhiêu? URL WEB: https://db.ptit.edu.vn/question-detail/285bb142-c6ab-429b-8117-3ac68fa9bfd1 URL API: https://dbapi.ptit.edu.vn/api/app/question/285bb142-c6ab-429b-8117-3ac68fa9bfd1 Loại Database: Mysql ------------------------------

Mô tả bảng

Bảng pub_sector

Bảng này lưu trữ thông tin về các tổ chức công, bao gồm tên tổ chức, ngân sách, và lĩnh vực hoạt động.

+----------+-----------+--------------------------------------------+
| Cột      | Kiểu dữ liệu | Mô tả                                   |
+----------+-----------+--------------------------------------------+
| name     | TEXT        | Tên tổ chức                              |
| budget   | INTEGER     | Ngân sách của tổ chức                    |
| sector   | TEXT        | Lĩnh vực hoạt động (ví dụ: public/private) |
+----------+-----------+--------------------------------------------+

Nhiệm vụ

Tính tổng ngân sách (SUM(budget)) dành cho các sáng kiến trong khu vực công (sector = 'public').

Input mẫu

+------------------+----------+----------+
| name             | budget   | sector   |
+------------------+----------+----------+
| National Library | 500000   | public   |
| City Hospital    | 1200000  | public   |
| Tech Academy     | 800000   | private  |
| Gov IT Center    | 950000   | public   |
| EduHub           | 400000   | private  |
+------------------+----------+----------+

Output mẫu

+-------------+
| SUM(budget) |
+-------------+
| 2650000     |
+-------------+
