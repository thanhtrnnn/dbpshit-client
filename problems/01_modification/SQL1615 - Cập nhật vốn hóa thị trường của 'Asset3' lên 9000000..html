ID: SQL1615 Tiêu đề: Cập nhật vốn hóa thị trường của 'Asset3' lên 9000000. URL WEB: https://db.ptit.edu.vn/question-detail/6b8671d6-ddd4-4646-9ff5-9442244d1830 URL API: https://dbapi.ptit.edu.vn/api/app/question/6b8671d6-ddd4-4646-9ff5-9442244d1830 Loại Database: Sql Server ------------------------------

Bảng `digital_asset` và `market_capitalization`:

  • `digital_asset`:
  • `id (INT)`: Mã số duy nhất cho tài sản kỹ thuật số.
  • `name (VARCHAR(255))`: Tên của tài sản kỹ thuật số.
  • `category (VARCHAR(255))`: Danh mục của tài sản kỹ thuật số (ví dụ: "Crypto", "Security").


  • `market_capitalization`:
  • `id (INT)`: Mã số duy nhất cho thông tin vốn hóa thị trường.
  • `asset_id (INT)`: ID của tài sản kỹ thuật số.
  • `value (INT)`: Giá trị vốn hóa thị trường của tài sản kỹ thuật số.


Nhiệm vụ

Cập nhật giá trị vốn hóa thị trường của tài sản kỹ thuật số của tài sản kỹ thuật số 'Asset3' là 9000000.


Input mẫu

Bảng digital_asset

+----+--------+-----------+
| id | name   | category  |
+----+--------+-----------+
| 1  | Asset1 | Crypto    |
| 2  | Asset2 | Security  |
| 3  | Asset3 | Crypto    |
| 4  | Asset4 | Security  |
+----+--------+-----------+

Bảng market_capitalization

+----+----------+---------+
| id | asset_id | value   |
+----+----------+---------+
| 1  | 1        | 5000000 |
| 2  | 2        | 7000000 |
| 3  | 3        | 8000000 |
| 4  | 4        | 6000000 |
+----+----------+---------+

Output mẫu

+----+----------+---------+
| id | asset_id | value   |
+----+----------+---------+
| 1  | 1        | 5000000 |
| 2  | 2        | 7000000 |
| 3  | 3        | 9000000 |
| 4  | 4        | 6000000 |
+----+----------+---------+
