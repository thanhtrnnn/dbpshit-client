Table: Student

+-----------+----------------+
| Column    | Type           |
+-----------+----------------+
| StudentID | int            |
| FullName  | varchar(100)   |
| Email     | varchar(100)   |
| BirthDate | date           |
| GPA       | decimal(3,2)   |
+-----------+----------------+

StudentID is the unique ID of each student and will be the primary key.  

FullName is the full name of the student, required for every record.  

Email is the student’s email address, also required.  

BirthDate is the student’s date of birth.  

GPA is the student’s grade point average on a 0.00–4.00 scale.  

Each row in this table represents one student and their basic academic information.


Task:  

Write an SQL statement to create the Student table with the columns above.  

Requirements:

- StudentID is the primary key.

- FullName and Email cannot be NULL.

- GPA cannot be NULL and should allow values like 3.50, 2.75, 4.00.
