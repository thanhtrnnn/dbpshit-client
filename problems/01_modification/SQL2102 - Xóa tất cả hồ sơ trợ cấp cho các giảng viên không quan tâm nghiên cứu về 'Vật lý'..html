Cho hai bảng dữ liệu:

Bảng grants

  • id (INT): Mã tài trợ duy nhất
  • title (TEXT): Tiêu đề tài trợ
  • amount (FLOAT): Số tiền tài trợ
  • faculty_name (VARCHAR(50)): Tên khoa nhận tài trợ

Bảng faculty

  • id (INT): Mã khoa
  • name (VARCHAR(50)): Tên khoa
  • research_interest (TEXT): Lĩnh vực nghiên cứu của khoa

Nhiệm vụ:

Xóa tất cả hồ sơ trợ cấp cho các giảng viên không quan tâm nghiên cứu về 'Vật lý'.
