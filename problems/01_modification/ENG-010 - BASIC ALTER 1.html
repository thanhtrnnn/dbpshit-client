Table: Students

+-------------+--------------+
| Column Name | Type         |
+-------------+--------------+
| student_id  | int          |
| full_name   | varchar(100) |
| birth_date  | varchar(20)  |
| email       | varchar(100) |
| phone       | varchar(20)  |
| address     | varchar(255) |
+-------------+--------------+

student_id stores the numeric ID of each student, currently without primary key or auto-increment.

full_name stores the student’s full name.

birth_date is stored as text for now, not as a real date type.

email stores the student’s email address and is currently nullable and shorter.

phone stores the student’s phone number and is currently nullable and short.

address stores the student’s home address.

Each row in this table represents one student and their basic contact information.


Task:

The Students table already exists with the structure above.  

You need to change it so that:

- student_id becomes an INT NOT NULL AUTO_INCREMENT and is the primary key.

- birth_date is stored as a proper DATE type.

- email becomes VARCHAR(255) and cannot be NULL.

- phone becomes VARCHAR(50) and cannot be NULL.

- a new column created_at is added, with type DATETIME and default value CURRENT_TIMESTAMP.


Write an ALTER TABLE statement to apply all these changes to the Students table.
