ID: SQL1830 Tiêu đề: Liệt kê tất cả các trạm có lối đi cho xe lăn trên các tuyến xe buýt và tàu điện ngầm. URL WEB: https://db.ptit.edu.vn/question-detail/bcebdca0-421c-4728-bff4-08eecaa2de1f URL API: https://dbapi.ptit.edu.vn/api/app/question/bcebdca0-421c-4728-bff4-08eecaa2de1f Loại Database: Mysql ------------------------------

Mô tả bảng

Bảng stations

Bảng này lưu trữ thông tin về các trạm, bao gồm ID trạm và tên trạm.

+-------------+-----------------------+---------------------------------------+
| Cột         | Kiểu dữ liệu          | Mô tả                                 |
+-------------+-----------------------+---------------------------------------+
| station_id  | INT                   | Mã định danh trạm                     |
| station_name| VARCHAR(255)          | Tên của trạm                          |
+-------------+-----------------------+---------------------------------------+

Bảng bus_stations

Bảng này lưu trữ thông tin về các trạm xe buýt, bao gồm ID tuyến, ID trạm, và hỗ trợ người khuyết tật.

+---------------------------+-----------------------+----------------------------------------------+
| Cột                       | Kiểu dữ liệu          | Mô tả                                        |
+---------------------------+-----------------------+----------------------------------------------+
| route_id                  | INT                   | Mã tuyến xe buýt                             |
| station_id                | INT                   | Mã trạm xe buýt                              |
| is_wheelchair_accessible  | BOOLEAN               | Trạm có hỗ trợ người khuyết tật không        |
+---------------------------+-----------------------+----------------------------------------------+

Bảng subway_stations

Bảng này lưu trữ thông tin về các trạm tàu điện ngầm, bao gồm ID tuyến, ID trạm, và hỗ trợ người khuyết tật.

+---------------------------+-----------------------+----------------------------------------------+
| Cột                       | Kiểu dữ liệu          | Mô tả                                        |
+---------------------------+-----------------------+----------------------------------------------+
| line_id                   | INT                   | Mã tuyến tàu điện ngầm                       |
| station_id                | INT                   | Mã trạm tàu điện ngầm                        |
| is_wheelchair_accessible  | BOOLEAN               | Trạm có hỗ trợ người khuyết tật không        |
+---------------------------+-----------------------+----------------------------------------------+

Nhiệm vụ

Liệt kê tất cả các trạm có lối đi cho xe lăn (is_wheelchair_accessible = TRUE)

trên các tuyến xe buýt hoặc tàu điện ngầm.

Input mẫu

Bảng stations

+-------------+--------------+
| station_id  | station_name |
+-------------+--------------+
| 1           | Central Park |
| 2           | City Hall    |
| 3           | Riverside    |
| 4           | North Gate   |
+-------------+--------------+

Bảng bus_stations

+----------+-------------+---------------------------+

+----------+-------------+---------------------------+
| 10       | 1           | TRUE                      |
| 11       | 3           | FALSE                     |
| 12       | 4           | TRUE                      |
+----------+-------------+---------------------------+

Bảng subway_stations

+----------+-------------+---------------------------+
| line_id  | station_id  | is_wheelchair_accessible  |
+----------+-------------+---------------------------+
| 1        | 2           | TRUE                      |
| 2        | 3           | TRUE                      |
| 3        | 4           | FALSE                     |
+----------+-------------+---------------------------+

Output mẫu

+--------------+
| station_name |
+--------------+

| Riverside    |
| North Gate   |
+--------------+
