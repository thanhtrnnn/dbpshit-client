URL WEB: https://db.ptit.edu.vn/question-detail/c4dade3f-2b0e-44f9-9f3a-6bcde2dad315 URL API: https://dbapi.ptit.edu.vn/api/app/question/c4dade3f-2b0e-44f9-9f3a-6bcde2dad315 Loại Database: Mysql ------------------------------

Bảng tangible_heritage lưu trữ thông tin về các di sản vật thể, bao gồm mã di sản, tên di sản và khu vực.

  • id (INT): Mã di sản
  • name (VARCHAR(255)): Tên di sản
  • region (VARCHAR(255)): Khu vực di sản

Bảng intangible_heritage lưu trữ thông tin về các di sản phi vật thể, bao gồm mã di sản, tên di sản và khu vực.

  • id (INT): Mã di sản
  • name (VARCHAR(255)): Tên di sản
  • region (VARCHAR(255)): Khu vực di sản

Nhiệm vụ:

Có bao nhiêu di sản (vật thể và phi vật thể) ở khu vực Amazon?
