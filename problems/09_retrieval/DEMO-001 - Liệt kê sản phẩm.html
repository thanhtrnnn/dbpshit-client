Bảng: PRODUCT

+------------+--------------+
| Column Name| Type         |
+------------+--------------+
| ProductID  | int          |
| Name       | varchar      |
| Price      | decimal(10,2)|
+------------+--------------+


Trong SQL, ProductID là khóa chính của bảng này.

ProductID là cột số nguyên định danh duy nhất mỗi sản phẩm.

Name là tên sản phẩm.

Price là giá của sản phẩm.


Nhiệm vụ:

Liệt kê các sản phẩm có giá lớn hơn 100.

Trả về bảng kết quả gồm các cột (ProductID, Name).


Ví dụ:

Input:

Bảng PRODUCT:

+-----------+-----------+--------+
| ProductID | Name      | Price  |
+-----------+-----------+--------+
| 1         | Chair     |  80.00 |
| 2         | Table     | 150.00 |
| 3         | Lamp      | 120.50 |
| 4         | Sofa      |  99.99 |
| 5         | Bookshelf | 250.00 |
| 6         | Cup       |  15.00 |
+-----------+-----------+--------+


Output:

+-----------+-----------+
| ProductID | Name      |
+-----------+-----------+
| 2         | Table     |
| 3         | Lamp      |
| 5         | Bookshelf |
+-----------+-----------+
