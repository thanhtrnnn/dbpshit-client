Table: Customer

+--------------+--------------+
| Column Name  | Type         |
+--------------+--------------+
| CustID       | int          |
| CustomerName | varchar(100) |
| City         | varchar(50)  |
+--------------+--------------+

CustID is the unique ID of each customer, used as the primary key.

CustomerName is the full name of the customer.

City is the city where the customer lives.

Each row in this table represents one customer.


Table: SaleOrder

+--------------+----------------+
| Column Name  | Type           |
+--------------+----------------+
| OrderID      | int            |
| CustID       | int            |
| OrderDate    | date           |
| TotalAmount  | decimal(10,2)  |
+--------------+----------------+

OrderID is the unique ID of each order, used as the primary key.

CustID is the ID of the customer who placed the order. CustID is a foreign key referencing CustID in the Customer table.

OrderDate is the date when the order was created.

TotalAmount is the total value of the order.

Each row in this table represents one order placed by a customer.


Task:

Write an SQL query to print OrderID, CustomerName, City, OrderDate, and TotalAmount for all orders where the customer lives in 'London' or 'Paris', and the order’s TotalAmount is greater than 200.


Example

Input:

Customer

+--------+--------------+---------+
| CustID | CustomerName | City    |

| 1      | Alice        | London  |
| 2      | Bob          | Paris   |
| 3      | Charlie      | Hanoi   |
+--------+--------------+---------+


SaleOrder

+---------+--------+------------+-------------+
| OrderID | CustID | OrderDate  | TotalAmount |
+---------+--------+------------+-------------+
| 101     | 1      | 2024-01-10 | 150.00      |
| 102     | 2      | 2024-01-12 | 220.50      |
| 103     | 3      | 2024-01-15 | 300.00      |
+---------+--------+------------+-------------+


Output:

+---------+--------------+---------+------------+-------------+
| OrderID | CustomerName | City    | OrderDate  | TotalAmount |

| 102     | Bob          | Paris   | 2024-01-12 | 220.50      |
