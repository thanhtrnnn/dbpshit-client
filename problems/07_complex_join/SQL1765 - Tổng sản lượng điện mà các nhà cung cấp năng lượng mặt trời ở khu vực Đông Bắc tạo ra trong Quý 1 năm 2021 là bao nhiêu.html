ID: SQL1765 Tiêu đề: Tổng sản lượng điện mà các nhà cung cấp năng lượng mặt trời ở khu vực Đông Bắc tạo ra trong Quý 1 năm 2021 là bao nhiêu? URL WEB: https://db.ptit.edu.vn/question-detail/8408aa41-2fcb-437d-9ecc-87dc2ab75d02 URL API: https://dbapi.ptit.edu.vn/api/app/question/8408aa41-2fcb-437d-9ecc-87dc2ab75d02 Loại Database: Mysql ------------------------------

Cấu trúc bảng:

Bảng region

  • id (INT): Mã khu vực
  • name (VARCHAR(50)): Tên khu vực

Bảng provider

  • id (INT): Mã nhà cung cấp
  • name (VARCHAR(50)): Tên nhà cung cấp
  • region_id (INT): Mã khu vực nơi nhà cung cấp hoạt động

Bảng renewable_source

  • id (INT): Mã nguồn năng lượng tái tạo
  • name (VARCHAR(50)): Tên nguồn năng lượng tái tạo (ví dụ: 'Solar')
  • provider_id (INT): Mã nhà cung cấp năng lượng

Bảng power_generation

  • id (INT): Mã sản lượng điện
  • date (DATE): Ngày sản xuất điện
  • source_id (INT): Mã nguồn năng lượng tái tạo
  • power_generated (INT): Sản lượng điện được tạo ra

Nhiệm vụ:

Tổng sản lượng điện mà các nhà cung cấp năng lượng mặt trời ở khu vực Đông Bắc tạo ra trong Quý 1 năm 2021 là bao nhiêu?
